# TUI Client

The TUI is a paired user client that talks to the API server.

## Key Behaviors

- stores its own user token
- stores its own active profile selection
- supports the full `/profile` command family
- is a good power-user fallback for profile management

## Good Fit

Use the TUI when you want:

- fast local iteration
- a lightweight paired client
- easy profile switching without the admin web UI
