# Menubar Client

The macOS menubar app is a paired user client for day-to-day chat.

## Key Behaviors

- stores its own user token
- stores its own active profile selection
- can differ from other clients for the same user
- supports the `/profile` command family
- also exposes profile switching through the UI

## Good Fit

Use the menubar app when you want:

- a private profile-focused chat surface
- quick profile switching
- a lightweight macOS-native client
