# Coding Agents (Claude Code, OpenAI Codex, Gemini CLI, ...)

- Instead of letting the main model work on complex coding tasks, use Coding Agents such as Claude Code or OpenAI Codex
- Codex is installed in the Sandbox Docker Image
- Pattern: Use tmux to create a persistent session
- Login on first use, your Skitter agent should guide you through it
- Skitter will interact with the Coding Agent
