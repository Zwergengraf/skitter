# macOS Conversation Mode (low-latency TTS and STT)

- Use macOS menubar app in Conversation Mode
- Tip: Local and small LLM for maximum speed, MoE or <= 32B parameters, e.g. gpt-oss-20b, Gemma 3 12B / 27B, ...
- Tip: Use local TTS (OpenAI API compatible), for example [Pocket TTS](https://github.com/kyutai-labs/pocket-tts) / [OpenAI API Wrapper](https://github.com/teddybear082/pocket-tts-openai_streaming_server)
